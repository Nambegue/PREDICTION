Instructions pour exécuter l'application :

1. Téléchargez et décompressez le dossier.
2. Ouvrez le fichier `index.html` dans un navigateur web.
3. Entrez les noms des deux équipes.
4. Cliquez sur "Prédire le résultat" pour voir la prédiction et les graphiques.

Remarque : Remplacez "VOTRE_CLE_API_FOOTBALL" dans `script.js` par votre clé API API-Football.
